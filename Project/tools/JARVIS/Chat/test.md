# 実装計画書：main.cpp 先頭コメントの変更

## 背景・目的
`Project\DirectXGame\main.cpp` の先頭行は現在、署名コメント `// J.A.R.V.I.S.` になっている。
これを `S.U.N.D.A.Y.` 表記へ変更する。動作には影響しない見た目（署名）の変更。

## 変更内容
| 項目 | 内容 |
| --- | --- |
| 対象ファイル | `Project\DirectXGame\main.cpp` |
| 対象行 | 1行目のみ |
| 変更前 | `// J.A.R.V.I.S.` |
| 変更後 | `// S.U.N.D.A.Y.` |

- 既存プロジェクトの命名 `S.U.N.D.A.Y.` に揃えたドット区切り表記を採用。
- `// sunday` など別表記を希望する場合は要相談。

## 影響範囲
- コメント1行のみのため、ビルド・実行への機能的影響はなし。

## 検証手順
1. 変更後に `CG2_0_1.sln` をビルドして通ることを確認。
2. 先頭行が意図通り `// S.U.N.D.A.Y.` になっていることを確認。
