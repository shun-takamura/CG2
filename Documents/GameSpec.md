## 2. シーン構成

```
Title  → Hub → StagePlay (道中STG → ボス戦シームレス) → Result → Hub
              ↑                                              ↓
              └──────────────────────────────────────────────┘
```

| シーン | 役割 |
|---|---|
| **Title** | 無操作10秒でデモプレイ映像（Media Foundation 動画）自動再生 |
| **Hub** | タブUI（ステージセレクト / スキルショップ / スキル装備 / タイトル / 終了）。シーン遷移を挟まずタブ切替 |
| **StagePlay** | 道中レールSTG (2-3分) → ボス戦3Dアクション (3-5分) をシームレスに繋ぐ |
| **Result** | スコア → SP獲得 → Hubへ戻る |
| **DemoScene** | （技術検証用 / 絶対削除禁止） |

## 3. プレイヤー仕様

### 3.1 ステータス
- **初期HP**: 100
- **被弾無敵**: 0.2秒
- **回復**: ジャスト回避1回につき1度、`X`ボタン/`R`キーで +5回復（コストなし、STG/ボス戦それぞれ10回まで）
- **射撃後隙**: 0（即回避・近接・移動可）

### 3.2 ゲームオーバー
- STG中死亡 → STG最初から
- ボス戦中死亡 → ボス戦開始時HPで再開
- ステージセレクトに戻る選択肢あり

### 3.3 デザイン
- 『パルテナの鏡』ピット相当の人型機体

## 5. 射撃 / チャージシステム

### 5.1 通常射撃
- **発射間隔**: 0.3秒
- **弾の判定**: 大きめ（ライトゲーマー向け）
- **後隙**: 0

### 5.2 チャージショット
- 「非射撃中」「マルチロック弾発射中ではない」時間で計測
- 通常射撃を1発でも撃つと**即0リセット**

| 段階 | 条件 | 効果 | 演出 |
|---|---|---|---|
| 1段階目 | 3秒 | 判定拡大 + 攻撃力UP | シリンダープリミティブ |
| 2段階目 | 6秒（さらに3秒） | ホーミング + 貫通付与 | 1段階目 + パーティクル大量 |

### 5.3 マルチロックオン
- **発動条件**: 非射撃中のみ
- **ロック方式**: 敵にレティクル0.3秒重ね続けでフラグON
- **上限**: 初期5（スキルで増加可）
- **発射**: 射撃ボタンを押した瞬間、全ロック対象へ一斉発射（チャージ状態を継承、発射後チャージ0）

### 5.4 精密射撃モード
- ホールド / トグル を起動前設定で選択
- カメラがプレイヤー右肩越し（Over the Shoulder）へ
- UI退避（HPバー等が画面外へスライドアウト）
- エイムアシスト（速度補正、コントローラ/マウス両対応）
- 弱点ハイライト
- 画面エフェクト: ヴィネット + 周辺ガウシアンぼかし
- マウスカーソルは ClipCursor + 非表示

## 6. 近接攻撃

| 種類 | キー | 性能 | 後隙 |
|---|---|---|---|
| 強 | Y / Q | 単発・高威力・モーション大 | 長い |
| 弱 | B / E | 最大3段コンボ、途中で射撃/回避に移行可 | 1-2段目=なし、3段目=少 |

3段目の合計威力 < 強1発

## 7. 必殺技（装備制：1つだけ持ち込み）

| 必殺技 | ゲージ | 性能 |
|---|---|---|
| ハイマットフルバースト | 100 | 全画面敵を順次ロック→一斉射撃。発動中無敵、5体上限を無視 |
| ディスラプター | 120 | 直線一閃、超高火力。敵動作停止+世界断裂演出。撃ち終わり後隙あり |

- ゲージは溜まった時点が上限（自動で 100/120 で頭打ち）
- ステージ開始時 0
- 弱コンボの方が貯まりやすい
- スキル装備画面で1つ選択（スキルコストとは別枠）
- 発動ボタンは1つ（R3 / F）

## 8. ジャスト回避

- **判定窓**: 100ms前後（要調整）
- **対象**: 敵の攻撃判定のみ。突進タックル等も対象。ただの移動接触は対象外
- **効果**: 3秒スロー（オブジェクト単位ポストエフェクト：プレイヤー+対象敵以外をグレースケール+ラジアルブラー）
- **必殺技ゲージ**: ジャスト回避でも蓄積
- **分身カウンター**: スロー中常時受付
  - **下/A/SPACE** = 追加回避（手前分身が実体化、無敵）
  - **左/Y/Q** = 強カウンター（奥分身、敵に肉薄して強攻撃）
  - **右/B/E** = 弱コンボカウンター（右分身、3段コンボ強制）

## 9. ボス戦（3Dアクション）

- **射撃可能**（ロックオンも可。ボス1体想定なのでマルチロックの恩恵小）
- **移動**: 地上360度自由、ジャンプなし、近接時自動踏み込み
- **回避**: バックステップ系モーション
- **カメラ**:
  - デフォルト: フリーカメラ
  - L1短押し: ターゲットカメラ（一瞬ボス向く）
  - L1を0.5秒長押し: ロック⇔フリー トグル
- **想定ボス数**: 1 (場合により2)

## 10. 敵ユニット（道中ザコ）

| 種別 | 特徴 |
|---|---|
| **ドローン機** | プレイヤー同等以上のサイズ、コライダー大きめ、直線弾 or 突撃、1-2発で撃破 |
| **運び屋（キャリアー）** | 攻撃しない、ザコを生成し続ける、HP極大。チャージ/近接/必殺技で削る対象 |

## 11. シーン遷移（道中→ボス戦シームレス着地）

- 道中最終地点でフライトカメラが地上スレスレまで降下
- プレイヤー機が地面に着地モーション
- カメラ制御が「レール追従」→「背面カメラ」へ数フレーム補間
- 暗転なしでボス戦突入

## 12. スキル / SP

- **獲得**: 敵撃破 + ジャスト回避数で加算
- **使用**: Hubで購入・装備
- **スキル種類**:
  - HP増加
  - 射撃間隔短縮
  - 攻撃力倍率（全攻撃に乗る）
  - マルチロック上限増加
  - 必殺技ゲージ蓄積効率
- **Lv1〜3** + コスト式（プレイヤーコスト上限内でビルド）
- 装備枠は無制限、コストで縛る

## 13. ステージ進行

- **目標5ステージ**（余裕があれば追加）
- 1ステージずつアンロック
- 1面はチュートリアル兼進行実感重視

## 14. ポーズメニュー

- ESC/START で開く
